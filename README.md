# Gravity Room - Professional Development Portfolio

Just as one gains strength through extreme, high-gravity training (up to 500x Earth's gravity) in the Capsule Corp gravity room, often focusing on intense, self-imposed isolation to surpass limits, this space represents continuous professional development and skill enhancement.

## About This Project

A professional portfolio website showcasing skills development in web security, featuring a modern violet-themed design.

## Features

- **Skills Development Section**: Highlighting expertise in web security including:
  - Burp Suite Certified Practitioner (BSCP) certification
  - Penetration Testing
  - Vulnerability Analysis
  - API Security

- **Modern Violet Design**: Clean, professional design with violet/purple color scheme (#7c3aed)
- **Responsive Layout**: Works seamlessly on desktop and mobile devices
- **Certifications Showcase**: Dedicated section for professional certifications and training

## How to View

Simply open `index.html` in your web browser, or serve it with any web server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve

# Then visit http://localhost:8080
```

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Responsive Design
- Modern UI/UX principles
